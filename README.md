# runtrackJs

La Runtrack JavaScript est un parcours progressif composé de plusieurs jours de pratique visant à maîtriser les bases du langage JavaScript ainsi que son utilisation dans le développement web.

Chaque jour introduit de nouveaux concepts et propose des exercices pratiques (jobs) pour les mettre en application.


## Objectifs pédagogiques
Comprendre les bases de JavaScript Manipuler le DOM Gérer les événements utilisateur Structurer du code propre et lisible Développer des interactions dynamiques sur une page web

## Organisation du projet
Le projet est structuré en plusieurs dossiers, chacun correspondant à un jour de la Runtrack. Chaque dossier de jour contient plusieurs sous-dossiers, chacun représentant un job (exercice) défini dans le sujet.

## Structure d'un Job
Chaque job doit respecter les consignes suivantes :

Contenir : index.html script.js Le fichier HTML doit : inclure une structure HTML valide importer le fichier JavaScript Le code doit être : clair Les fichiers doivent contenir uniquement ce qui est demandé

## Méthodologie 
Avancer job par job Tester chaque fonctionnalité dans le navigateur Utiliser la console pour déboguer Faire des commits réguliers avec des messages explicites

🔹 Jour 03

🔹 Jour 04

🔹 Jour 05

(Les jours suivants continuent avec des notions plus avancées)

## Technologies utilisées
HTML5 CSS3 JavaScript (Vanilla JS)

## Lancer le projet
Cloner le repository Ouvrir un dossier de job Lancer le fichier index.html dans un navigateur Ouvrir la console (F12) pour voir les résultats

## Progression
Ce projet est conçu pour être suivi de manière progressive. Chaque jour s’appuie sur les connaissances acquises précédemment.
